Auth System

This is a secure authentication system where users can log in and access services through multiple authentication methods. It includes features like JWT with public/private keys, bcrypt for password hashing, Redis caching for sessions, password reset via email OTP, and Single Sign-On (SSO) with Google. The system is containerized with Docker, orchestrated with Kubernetes, ensuring scalability and reliability.

Node.js Express.js TypeScript Redis JWT Elastic Stack Docker MySQL bcrypt Email OTP SSO (Google) Public/Private Keys

My Role :

  • Developed authentication & authorization with JWT and bcrypt
  • Implemented password reset with email OTP functionality
  • Integrated Google SSO for seamless login
  • Used Redis for session caching and scalability
  • Containerized services with Docker
  • Created a single docker-compose.yml to install Redis, MySQL, Elasticsearch, Logstash, and Kibana
  • Configured centralized logging to send application logs from containers to Elasticsearch via Logstash
